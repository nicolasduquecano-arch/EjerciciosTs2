# Taller 2

Proyecto de ejemplo en TypeScript que incluye varias aplicaciones sencillas como un cajero automático, un sistema de turnos, una boletería y un carrito de compras.

## Instalación

Instala las dependencias del proyecto con npm:

```bash
npm install
```

## Ejecución

Compila y revisa el proyecto con TypeScript:

```bash
npx tsc --noEmit
```

Ejecuta el archivo principal directamente con Node:

```bash
node index.ts
```

Si prefieres usar Bun, puedes ejecutar:

```bash
bun run index.ts
```
